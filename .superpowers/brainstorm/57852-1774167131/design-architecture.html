Design — Ristorante Demo

Sezione 1 di 3 — Architettura generale. Dicci se va bene prima di procedere.

Il progetto

Landing page + sistema prenotazioni per ristorante fittizio "Osteria Bellavista"

Un demo realistico che un ristoratore del Luganese potrebbe usare dal giorno 1.

Struttura — Frontend (React)

React SPA — Sezioni della pagina
🖼️ Hero — Foto ristorante, nome, tagline, CTA "Prenota un tavolo"
🍽️ Menu — Antipasti / Primi / Secondi / Dolci (statico, visivo)
📸 Gallery — Griglia foto piatti e ambiente
📅 Booking Widget — Il cuore del progetto (vedi flusso sotto)
📍 About + Contatti — Storia, indirizzo, orari, Google Maps embed

Struttura — Backend (FastAPI + SQLite)

3 endpoint — semplice e sicuro
GET /slots?date=2026-03-22
Ritorna gli orari disponibili per la data scelta
POST /bookings
Crea prenotazione (status: pending) + invia email di conferma con token
GET /confirm/{token}
L'utente clicca il link → status diventa "confirmed" → notifica al ristorante
